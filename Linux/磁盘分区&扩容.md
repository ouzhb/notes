# 分区

    # 进入某硬盘分区页面
    fdisk /dev/sda

    # 输入以下命令
        n：创建分区
        d：删除分区
        p：分区信息
        w：保存退出

        注意直接创建的主分区可以直接使用，扩展分区不能直接使用，需要在上面创建逻辑分区

        partprobe
